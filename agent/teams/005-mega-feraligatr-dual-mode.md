## Playing this Team

### Primary Mode
Sableye, Volcarona, Feraligatr, Kingambit
Game Plan: Use Fake Out and redirection (Volcarona/Sableye) to help Feraligatr set up Dragon Dance, while Kingambit provides strong priority and cleanup.

### Conditional Mode 1
Sinistcha, Kingambit, Sableye, Gardevoir (or Feraligatr)
Use When: The opponent relies heavily on Tailwind or fast, frail attackers.
Game Plan: Set up Trick Room with Sinistcha and sweep with slow Kingambit/Gardevoir.

## Analysis

### Strengths
- Dual-mode speed control with Tailwind (Volcarona) and Trick Room (Sinistcha).
- Excellent setup opportunities for Feraligatr with Fake Out (Sableye) and Rage Powder (Volcarona).
- Strong anti-Intimidate option in Kingambit (Defiant).

### Weaknesses
- Heavy reliance on Focus Sash for Sableye and potential vulnerability to spread moves.
- Feraligatr is vulnerable to Intimidate cycling if Kingambit is not present.

### Roles
- Speed control: Tailwind, Trick Room, Quash.
- Intimidate / Attack drops: None. (Consider adding an Intimidate user).
- Redirection: Volcarona, Sinistcha.
- Fake Out: Sableye.
- Setup: Feraligatr.
- Spread damage: Volcarona, Sinistcha, Gardevoir.
- Weather/Terrain: None.
