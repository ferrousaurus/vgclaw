# Valid Data

You MUST ONLY use the Pokemon in `assets/pokemon.json`, abilities in `assets/abilities.json`, moves in `assets/moves.json`, and held items in `assets/items.json`.

Pokemon MUST ONLY have one of the abilities listed to it, and MUST ONLY know moves present on its move list.
