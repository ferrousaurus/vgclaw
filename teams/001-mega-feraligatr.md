## Evaluation

This looks like a hyper offense team built around `Sableye + Mega Feraligatr`, with `Mega Gardevoir` as an alternate Mega mode. The analysis below ignores the `Feraligatrite` naming issue and evaluates the intended Mega Feraligatr game plan.

### Summary Verdict

**Team: Hyper Offense built around Sableye + Mega Feraligatr**

**Strengths**

- Strong setup support for Feraligatr: `Fake Out`, `Quash`, `Helping Hand`, and `Rage Powder` all buy turns for `Dragon Dance`.
- Good role compression: `Volcarona` provides both redirection and Tailwind, while `Arcanine` compresses `Intimidate + Snarl`.
- Real alternate mode exists: `Mega Gardevoir` gives a spread-damage path that does not rely on setting up Feraligatr.

**Top Issues**

- The team leans heavily on `Volcarona` for true speed control. If Tailwind is denied, the roster becomes mostly mid/slow speed with only `Quash` as backup.
- `Garchomp`, rain offense, and bulky Water/Electric pivots all pressure the same fault lines at once.
- Several good modes share the same support pieces, so disruption on `Sableye` or `Volcarona` can flatten multiple game plans.

### 1. Type Coverage

**Findings**

- Defensively, no single type hits 3 or more team members super effectively.
- The main pressure clusters are `Rock` into `Volcarona + Arcanine`, `Ground` into `Arcanine + Kingambit`, and `Water` into `Volcarona + Arcanine`.
- `Fighting` is mostly contained because `Sableye` is immune and `Gardevoir` resists it heavily, which matters for protecting `Kingambit`.
- Offensively, the team covers most important types through `Fire`, `Ground`, `Water`, `Steel`, `Psychic`, `Electric`, `Dark`, and Fairy pressure from Mega Gardevoir's `Hyper Voice`.
- The biggest practical offensive hole is into Dragon- and Ground-based attackers when `Mega Gardevoir` is not the active Mega. `Feraligatr` currently lacks `Ice Punch`, so `Garchomp` is answered mostly by `Gardevoir` mode and positioning.

**Problems Flagged**

- `Rock` and `Ground` pressure line up well into common `Volcarona + Arcanine + Kingambit` structures.
- The team does not have a clean non-Gardevoir answer to `Garchomp`-style threats.

**Tiered Fix Suggestions**

1. Move change: `Feraligatr: Double-Edge -> Ice Punch` if you want a cleaner `Garchomp` and Dragonite answer without needing Mega Gardevoir.
2. Item change: `Arcanine: Sitrus Berry -> Shuca Berry` if Ground pressure is the matchup you most want to patch.
3. Pokemon swap: if `Arcanine` keeps feeling redundant with `Volcarona`, replace it with `Pelipper`, `Sinistcha`, `Whimsicott`, or `Incineroar` depending on whether you want rain/Tailwind utility, bulkier redirection, faster Prankster speed control, or better pivot support.

### 2. Pair Synergies

**Top Synergy Pairs**

- `Sableye + Feraligatr`: elite `Fake Out + Dragon Dance` pair. `Helping Hand` also turns boosted `Liquidation` into immediate KO pressure, and `Quash` patches Feraligatr's mid-tier Speed when it does not boost.
- `Volcarona + Feraligatr`: `Rage Powder + Dragon Dance` is the team's second setup lane, and `Tailwind` gives Feraligatr a lower-risk speed route when setup is not safe.
- `Sableye + Mega Gardevoir`: `Helping Hand` and `Quash` make `Hyper Voice` much harder to position around, giving the alternate Mega mode real burst.

**Anti-Synergy Flags**

- `Volcarona + Arcanine` is the clearest field anti-synergy. They stack `Rock` and `Water` issues while both occupying similar Fire-type pressure space.
- `Arcanine + Kingambit` is manageable, but it creates a narrow corridor into `Ground` attacks if they share the field without strong offensive pressure.

**Missing Synergy Gaps**

- The team has good setup enablement, but the alternate Mega Gardevoir mode is less layered than the Feraligatr mode. It uses support well, but it does not have the same level of built-in protection or backup speed control.
- There is no spread-move immunity engine or weather engine, which is fine for this archetype.

**Tiered Fix Suggestions**

1. Move change: `Gardevoir: Psychic -> Icy Wind` if you want the Mega Gardevoir mode to create its own speed support instead of borrowing everything from `Volcarona` and `Sableye`.
2. Item change: keep `Charti Berry` on `Volcarona`; it is doing real work for the pair structures that otherwise fold to `Rock Slide`.
3. Pokemon swap: if you want a more robust alternate Mega mode, `Sinistcha`, `Whimsicott`, or `Incineroar` in the `Arcanine` slot each add a stronger support layer than the current double-Fire structure.

### 3. Bring-4 Modes

**Mode 1: Primary Mega Feraligatr mode**

- `Sableye / Volcarona / Feraligatr / Kingambit`
- Game plan: enable `Dragon Dance` with either `Fake Out` or `Rage Powder`, then clean with boosted Mega Feraligatr and `Sucker Punch` endgame.

**Validation**

- Speed control: yes, via `Tailwind` and `Quash`.
- Win condition: yes, clear setup sweeper mode.
- Pair synergy: excellent.
- Type gap warning: still soft into strong `Rock Slide` and fast `Ground` pressure if `Volcarona` is forced to lead.

**Mode 2: Alternate Mega Gardevoir mode**

- `Sableye / Volcarona / Gardevoir / Kingambit`
- Game plan: Mega evolve `Gardevoir`, pressure with `Hyper Voice`, and use `Quash` or `Tailwind` to control pace while `Kingambit` cleans.

**Validation**

- Speed control: yes, but still mostly borrowed from `Volcarona`.
- Win condition: yes, spread pressure plus endgame priority.
- Pair synergy: good, especially `Sableye + Gardevoir`.
- Type gap warning: this mode is more stable into Dragons, but still not especially resilient if `Volcarona` is removed early.

**Mode 3: Bulky disruption mode**

- `Arcanine / Volcarona / Gardevoir / Kingambit`
- Game plan: use `Intimidate + Snarl + Tailwind` to drag the game down and let `Mega Gardevoir` or `Kingambit` close.

**Validation**

- Speed control: only `Tailwind`.
- Win condition: present, but slower and less clean than the first two modes.
- Pair synergy: acceptable, but this is the least cohesive mode.
- Flag: this mode is playable, but it is not as structurally strong as the main two because it still depends on `Volcarona` and does not create much immediate turn-one pressure.

**Matchup Mapping**

- Against `Incineroar` balances: bring Mode 1. `Feraligatr` punishes Incineroar directly, and `Kingambit` threatens Defiant punishment if Intimidate is used recklessly.
- Against Dragon/Ground offense such as `Garchomp`: bring Mode 2. Mega Gardevoir's Fairy spread pressure gives you a much cleaner line than asking Feraligatr to brute-force through it.
- Against slower physical offense: bring Mode 3 only if you specifically want `Intimidate + Snarl` attrition. It is more reactive than proactive.
- Against rain: neither mode is perfect. Mode 2 is usually the better starting point because `Thunderbolt` gives at least one direct Water punish.

**Mode Coverage Gap**

- The team does not have a truly independent mode that functions well when `Volcarona` is unavailable.

**Tiered Fix Suggestions**

1. Move change: `Gardevoir: Psychic -> Icy Wind` for a second speed-control button that improves both Mode 2 and Mode 3 immediately.
2. Item change: no urgent item fix beyond matchup-specific berry tuning.
3. Pokemon swap: if you want a cleaner third mode, replace `Arcanine` with `Whimsicott`, `Pelipper`, `Sinistcha`, or `Incineroar` depending on whether you value Prankster speed control, weather/Tailwind compression, bulkier support, or stronger pivot utility.

### 4. Meta Threat Matchups

Current meta headliners from Pikalytics include `Incineroar`, `Sneasler`, `Garchomp`, `Sinistcha`, `Kingambit`, `Basculegion`, `Whimsicott`, `Charizard`, `Rotom-Wash`, `Pelipper`, and `Tyranitar`.

**Good or Solid Matchups**

- `Incineroar`: generally good. `Feraligatr` pressures it hard, `Kingambit` threatens Defiant, and `Arcanine` can at least keep the board stable with `Snarl`.
- `Sneasler`: manageable. `Arcanine` helps with `Intimidate`, `Gardevoir` threatens back well, and `Sableye` can disrupt pacing.
- `Tyranitar`: usually workable because `Feraligatr` pressures it directly and `Kingambit` can fight effectively into sand endgames.

**Shaky Matchups**

- `Garchomp`: one of the bigger structural issues. It pressures `Arcanine`, `Kingambit`, and `Volcarona` at once and can force awkward Mega choices.
- `Pelipper + Basculegion` rain: probably the cleanest stress test into this team. Rain attacks punish both Fire-types, and Tailwind wars are uncomfortable when your only direct Electric move is on Gardevoir.

**Tiered Fix Suggestions**

1. Move change: `Feraligatr: Double-Edge -> Ice Punch` to improve the `Garchomp` matchup.
2. Move change: `Gardevoir: Psychic -> Energy Ball` is a reasonable rain patch if those matchups matter more than Fighting- and Poison-targeting coverage.
3. Pokemon swap: if rain remains a major issue, consider `Pelipper` or `Sinistcha` over `Arcanine`. Both improve Water matchups in different ways.

### 5. Role Checklist

- Speed control: yes (`Tailwind`)
- Intimidate / attack drops: yes (`Intimidate`, `Snarl`, `Will-O-Wisp`)
- Redirection: yes (`Rage Powder`)
- Fake Out: yes (`Sableye`)
- Setup: yes (`Dragon Dance`)
- Spread damage: yes (`Heat Wave`, Mega Gardevoir `Hyper Voice`)
- Weather/Terrain: not needed for this archetype

**Assessment**

- The team covers the important hyper offense support roles.
- The main role gap is not missing speed control entirely, but lacking a second reliable form of it.
- The team also lacks a Ground immunity, which is not a formal role gap but matters structurally.

**Tiered Fix Suggestions**

1. Move change: `Gardevoir -> Icy Wind` or `Trick Room` over one coverage slot if you want backup speed control without changing the roster.
2. Pokemon swap: `Whimsicott` over `Arcanine` gives the team a much cleaner second speed-control slot, at the cost of some defensive utility.

### 6. Set Optimization

**Sableye**

- The set is coherent. `Focus Sash` matches its lead utility, and skipping `Protect` is fine on a `Fake Out` support.
- The fourth support move is the flexible slot. `Will-O-Wisp` is good into physical offense, but `Taunt` would improve anti-setup and anti-Trick Room lines.

**Volcarona**

- Very coherent support set. `Charti Berry` is justified because it patches the exact Rock weakness that would otherwise stop `Rage Powder` and `Tailwind` turns.
- No urgent change needed.

**Arcanine**

- The set is internally consistent as a bulky special utility piece.
- The biggest question is role overlap, not move legality. `Flamethrower + Scorching Sands + Snarl + Protect` is fine, but it does little to solve the Water and Ground matchups that pressure the team.

**Kingambit**

- The set is clean and low-maintenance. `Black Glasses` fits its endgame role, and `Protect` is good here.
- If the team wanted a greedier angle, `Swords Dance` would be the obvious fourth-move pressure upgrade, but the current set is safer.

**Feraligatr**

- `Protect + Liquidation + Dragon Dance` is the correct shell.
- `Double-Edge` makes sense only if Mega Feraligatr is the intended primary plan, because `Dragonize` turns it into strong Dragon pressure.
- The trade-off is coverage. Without `Ice Punch`, the set is narrower into Dragons and Ground-types.

**Gardevoir**

- `Protect + Hyper Voice + Thunderbolt` is strong on Mega Gardevoir.
- `Psychic` is playable, but it is the most replaceable move if you want either `Icy Wind`, `Trick Room`, `Moonblast`, or `Energy Ball` depending on what matchup you want to patch.

**EVs / Natures**

- The paste does not include EVs or natures, so exact benchmark recommendations cannot be made yet.
- If you add spreads later, this is the short benchmark list worth testing first:
  - `Feraligatr`: enough Speed to clear key mid-tier targets after `Dragon Dance`, then bulk.
  - `Gardevoir`: enough Speed to capitalize on Mega 100 base Speed cleanly, unless you add `Icy Wind` or `Trick Room` and want a different benchmark.
  - `Arcanine`: bulk-first with enough Special Attack to keep `Snarl` and `Scorching Sands` meaningful.

**Tiered Fix Suggestions**

1. Move change: `Gardevoir: Psychic -> Icy Wind` for better speed control.
2. Move change: `Sableye: Will-O-Wisp -> Taunt` if you want stronger anti-support lines.
3. EV/Nature adjustment: defer until spreads are added.

### 7. Win Condition Assessment

**Identified Win Conditions**

- `Setup sweeper`: `Mega Feraligatr` after `Dragon Dance`, enabled by `Sableye` or `Volcarona`.
- `Spread pressure`: `Mega Gardevoir Hyper Voice` plus `Volcarona Heat Wave`.
- `Single-target burst / cleanup`: `Kingambit` endgame with `Sucker Punch` and strong Dark STAB.
- `Attrition elements`: `Arcanine` provides a partial fallback with `Intimidate + Snarl`, but this is clearly secondary.

**Quality Assessment**

- `Mega Feraligatr` is powerful but high-dependency. It needs the Mega slot, a setup turn, and at least one support piece to function cleanly.
- The `Mega Gardevoir` mode is lower dependency and comes online faster, but it still leans on `Volcarona` or `Sableye` to stay ahead of faster teams.
- `Kingambit` is a good closer, but not a standalone primary plan.

**Sufficiency**

- The team does have at least two real win conditions, which is enough.
- The issue is independence. Both primary plans lean on the same support shell, so the team is less redundant than it first appears.

**Tiered Fix Suggestions**

1. Move change: `Gardevoir: Psychic -> Icy Wind` to make the alternate Mega mode more self-sufficient.
2. Move change: `Sableye: Will-O-Wisp -> Taunt` if your main failure point is opposing disruption rather than physical damage.
3. Pokemon swap: if you want a more independent backup win condition, replace `Arcanine` with a support that creates a separate engine, such as `Whimsicott`, `Pelipper`, or `Sinistcha`.

### 8. Lead & Resilience Check

**Lead Evaluation**

- `Sableye + Feraligatr`: the best default lead. `Fake Out + Dragon Dance` is high-impact, and it has fallback lines through `Quash` or immediate attacking.
- `Volcarona + Feraligatr`: strong but more brittle. `Rage Powder` is excellent into single-target disruption, but `Rock Slide` leads punish it heavily.
- `Sableye + Gardevoir`: very strong alternate lead. It projects immediate damage and useful disruption at the same time.
- `Volcarona + Gardevoir`: good into teams that respect redirection too much, but again very dependent on `Volcarona` surviving.

**Plan B Resilience**

- If `Sableye` is KOed early, the Feraligatr setup mode gets much worse but the Mega Gardevoir mode still functions.
- If `Volcarona` is removed early, the team loses its only true speed control. This is the biggest resilience issue on the roster.
- If setup is denied, the team can still win through Mega Gardevoir spread pressure and Kingambit cleanup, so it is not a one-plan team.
- Weather overwrite is irrelevant here, but opposing Tailwind and rain turns are both real strain points because of how much they punish the Fire-heavy support core.

**Tiered Fix Suggestions**

1. Move change: `Gardevoir: Psychic -> Icy Wind` is again the cleanest resilience upgrade.
2. Move change: `Sableye: Will-O-Wisp -> Taunt` if the main problem is opposing setup, Trick Room, or support mirrors.
3. Pokemon swap: replace `Arcanine` with `Whimsicott`, `Pelipper`, or `Sinistcha` if you want a stronger fallback mode when `Volcarona` is denied.

### Arcanine Replacements

If you are removing `Arcanine`, these are the best legal replacements for this team shell.

**1. Whimsicott**

- Best if you want the cleanest structural fix.
- Adds `Prankster Tailwind`, `Encore`, `Taunt`, and `Moonblast` while giving the team a second real speed-control piece.
- Improves the matchup spread against faster teams and makes the Mega Gardevoir mode less dependent on `Volcarona`.
- Trade-off: you lose `Intimidate`, `Snarl`, and some defensive padding.

**2. Pelipper**

- Best if you want to patch rain and Ground pressure at the same time.
- Adds `Drizzle`, `Tailwind`, `Wide Guard`, `Hydro Pump`, and `Icy Wind`.
- Gives `Feraligatr` stronger Water pressure without needing to set up as often, and it helps into `Rock Slide` modes through `Wide Guard`.
- Trade-off: it changes the team from pure hyper offense into a looser hybrid with weather texture, and it overlaps somewhat with `Volcarona` as another Tailwind support.

**3. Sinistcha**

- Best if you want the most durable support slot.
- Adds `Rage Powder`, `Matcha Gotcha`, `Strength Sap`, and `Trick Room` access.
- Improves Water matchups, gives the team a second redirector, and makes the board much less fragile when `Volcarona` is pressured.
- Trade-off: it is slower and more passive than the current slot, so the team becomes less explosive.

**4. Incineroar**

- Best if you want the highest floor and strongest pivoting utility.
- Adds `Fake Out`, `Parting Shot`, `Snarl`, and `Intimidate` in one slot.
- Makes your support shell extremely consistent and gives `Feraligatr` even more setup help.
- Trade-off: it overlaps with `Sableye` on Fake Out support and does less to solve the team's Water and Ground issues than `Pelipper` or `Sinistcha`.

**My Order**

1. `Whimsicott` if you want the cleanest upgrade to the team's structure.
2. `Sinistcha` if you want the safest support and best durability.
3. `Pelipper` if rain and Ground matchups are the priority.
4. `Incineroar` if you want raw consistency and do not mind keeping a support-heavy shell.

### Bottom Line

The team has a real offensive identity and two workable Mega modes. The core engine of `Sableye + Feraligatr` is good, and `Mega Gardevoir` gives the build a legitimate alternate route instead of a throwaway second Mega slot.

The main structural issue is not raw power. It is support concentration. Too many lines depend on `Volcarona` for speed control and on the same small group of support pieces to make the damage dealers function cleanly.

If you want the smallest, highest-impact change without changing the roster, the first thing to test is:

- `Gardevoir: Psychic -> Icy Wind`

If you want the second-smallest change aimed at `Garchomp` and Dragon matchups, test:

- `Feraligatr: Double-Edge -> Ice Punch`

If you want to keep the Mega Feraligatr identity but make the team more resilient overall, `Arcanine` is the right cut.

My first test would be:

- `Arcanine -> Whimsicott`

If you want the bulkier version instead, test:

- `Arcanine -> Sinistcha`

### Accepted Change

Accepted swap:

- `Arcanine -> Sinistcha`
- `Sinistcha ability: Hospitality` locked in over `Heatproof`
- `Sinistcha set: Matcha Gotcha / Rage Powder / Strength Sap / Trick Room`
- `Sinistcha item: Sitrus Berry`
- `Sableye: Will-O-Wisp -> Taunt`
- `Feraligatr: keep Double-Edge`

Rejected for now:

- `Kingambit: Protect -> Swords Dance`
- `Gardevoir: Psychic -> Icy Wind`

Why this fits the team:

- `Trick Room` gives the team a second speed-control mode for `Feraligatr`, `Kingambit`, and `Sableye`.
- `Hospitality` gives `Feraligatr` meaningful sustain on switch-in, which is especially useful if you keep `Double-Edge`.
- `Rage Powder` gives you a second redirector, so the team is less dependent on `Volcarona` surviving.
- `Strength Sap` gives `Sinistcha` a real recovery loop after soaking hits with `Rage Powder`.
- Redirection redundancy across `Sinistcha` and `Volcarona` gives you much cleaner bring-4 flexibility.
- The slot improves Water matchups and removes the awkward double-Fire overlap from the original build.

Main trade-off:

- You lose `Intimidate + Snarl`, so the team becomes less generically forgiving into strong physical offense and special spread pressure.

### Follow-Up Tuning After Sinistcha

Now that `Sinistcha` is locked in, the rest of the team wants to be tuned a little differently than the original version.

**What changed structurally**

- `Double-Edge` on `Feraligatr` got better, not worse. `Hospitality` makes the recoil more manageable, so it is no longer the first move I would cut.
- `Gardevoir: Psychic -> Icy Wind` is still good, but it is no longer the highest-priority follow-up because the team already gained a second speed-control mode through `Trick Room`.
- `Kingambit` got better because `Sinistcha` gives it both `Rage Powder` support and a real `Trick Room` environment.
- `Sableye` no longer has to carry as much of the defensive disruption load by itself, so its utility slot can be more proactive.

**Current priority tests**

1. `Sableye: Will-O-Wisp -> Taunt` - accepted

- Best if you want to improve the team's support mirror, anti-setup, and anti-Trick Room play.
- With `Strength Sap` now on `Sinistcha`, the team has another way to blunt physical attackers, so `Will-O-Wisp` is less mandatory than it was before.

2. `Kingambit: Protect -> Swords Dance`

- Best if you want to lean into the new `Sinistcha` mode.
- `Rage Powder + Trick Room` creates cleaner setup windows for `Kingambit` than the original `Arcanine` build did.
- Trade-off: you lose a generally safe doubles option, so this is stronger when you are committed to an aggressive board plan.
- Current decision: rejected for now. `Protect` keeps `Kingambit` more flexible across both Tailwind and Trick Room games.

3. `Gardevoir: Psychic -> Icy Wind`

- Still the best no-roster-change way to make the Mega Gardevoir mode less dependent on `Volcarona`.
- This is now more of a refinement than a structural fix.
- Current decision: rejected for now. The team already has enough speed-control texture through `Quash`, `Tailwind`, and `Trick Room`.

4. `Feraligatr: Double-Edge -> Ice Punch`

- Still the cleanest direct patch for `Garchomp` and Dragon-heavy matchups.
- With `Hospitality` in the build, I would keep `Double-Edge` unless those specific matchups keep causing problems.
- Current decision: rejected for now. `Double-Edge` is preferred with `Hospitality` support from `Sinistcha`.

**Current recommendation order**

If I were tuning the remaining five around the new `Sinistcha` version, I would test changes in this order:

1. Keep `Kingambit` as is with `Protect`
2. Keep `Gardevoir` as is without `Icy Wind`
3. Keep `Feraligatr` on `Double-Edge`

### EV Assignment

These are the recommended starting spreads for the finalized version of the team.

**Sableye**

- `252 HP / 4 Def / 252 SpD`, `Careful`
- Because `Prankster` and `Fake Out` handle most of its speed needs, Sableye wants bulk more than Speed.
- Max special bulk helps it keep functioning after `Focus Sash` is broken by spread moves like `Hyper Voice`, `Heat Wave`, or `Muddy Water`.

**Volcarona**

- `252 HP / 4 Def / 252 Spe`, `Timid`
- This keeps Tailwind as reliable as possible while still maximizing overall bulk.
- Max Speed matters more on this support set than extra Special Attack because its first job is getting `Tailwind` or `Rage Powder` off on time.

**Sinistcha**

- `252 HP / 156 Def / 100 SpD`, `Sassy`, `0 Spe IV`
- After the second pass, a mixed-bulk spread fits the role better than max special bulk. Sinistcha is often the Pokemon actually eating contact and spread damage while redirecting, so extra Defense matters in practice.
- The spread still preserves enough special bulk to handle repeated spread pressure with `Sitrus Berry`, while making `Rage Powder + Strength Sap` turns much more stable into physical boards.
- Minimum Speed remains important so Sinistcha stays a true `Trick Room` support piece.

**Kingambit**

- `252 HP / 252 Atk / 4 SpD`, `Brave`, `0 Spe IV`
- This is the cleanest fit for your current build: maximum power, maximum bulk, and full commitment to the `Trick Room` mode.
- `Protect` plus `Sucker Punch` already gives it enough flexibility outside of `Trick Room`, so there is little reason to invest in Speed.
- After the second pass, I would not get cute here. Kingambit is one of the few members that genuinely wants the straightforward max HP / max Atk spread.

**Feraligatr**

- `252 HP / 132 Atk / 124 Spe`, `Adamant`
- `124 Spe EVs` is the exact benchmark here. It reaches `114 Speed`, which lets Feraligatr outspeed max Speed `Jolly Garchomp` after one `Dragon Dance`.
- The remaining EVs go into HP first and then Attack, because this version of the team wants Feraligatr to survive setup turns and leverage `Double-Edge` with `Hospitality` support.
- Important benchmark: even `0 Atk Adamant` Feraligatr already OHKOs `252 HP Incineroar` with `Liquidation` after one `Dragon Dance`, so you can afford to invest meaningfully in bulk.

### Second-Pass Refinement

The tighter benchmark pass on the three most mode-sensitive Pokemon led to these final calls:
